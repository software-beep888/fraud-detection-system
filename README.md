# Fraud Detection System

A Python-based fraud detection system with **FastAPI** and **machine learning** for real-time predictions. This project uses an **Isolation Forest model** to detect potential fraudulent transactions and exposes a **REST API** for easy integration.

---

## Features

- Train and save a fraud detection model using **Isolation Forest**.
- FastAPI server for real-time predictions.
- `/predict` endpoint accepts transaction data and returns fraud prediction.
- `/` root endpoint confirms the API is running.
- Ready-to-use example for integration or portfolio showcase.
